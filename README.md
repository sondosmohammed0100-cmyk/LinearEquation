# 🧮 Linear Equation System - C++

A C++ console application built using **Microsoft Visual Studio Community 2026 (Stable 18.2.1)**.  
The project allows users to create, manage, and manipulate linear equations through a command-based interface.

---

## 📖 📌 Project Overview

This project implements a `linearEqu` class that represents a linear equation containing:

- 🔢 Variables  
- 📊 Coefficients  
- ➗ Constant term  

The system allows working with multiple equations and performing advanced operations on them.

---

## 📌 Example Commands
```
num_vars
equation 3
column x2
add 1 2
subtract 1 3
substitute x 1 2
quit to exit the program
```
